# 🎨 Identidade Visual — Schema e Template

> **Esta pasta NÃO contém identidade visual de nenhum produto.**
> A identidade visual real vem do `{REPO_DESTINO}/design_spec.md`.

## O que é esta pasta

Esta pasta contém **templates e exemplos** que mostram a estrutura esperada de um `design_spec.md` bem feito. Ela serve como referência para quem está escrevendo o arquivo de identidade visual de um novo produto.

## Como usar

1. Consulte o `design_spec_template.md` para entender a estrutura esperada
2. Crie o seu `design_spec.md` no repositório do produto seguindo o template
3. O squad lê o `design_spec.md` do repo do produto — não desta pasta

## Validação

O contrato de validação completo está em `shared/schemas/input-contracts.md`, seção "design_spec.md".
