# Skill — testability-advisor

## Objetivo
Orientar Back End e Front End para escrever código testável.

## Princípios de Testabilidade
- Funções puras sempre que possível
- Dependências injetadas, não instanciadas internamente
- IDs e data-attributes nos elementos HTML para fácil seleção em E2E
- APIs com respostas determinísticas para fixtures de teste
