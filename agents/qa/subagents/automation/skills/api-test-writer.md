# Skill — api-test-writer (QA)

## Objetivo
Suite de testes automatizados para todos os endpoints da API do o produto.

## Cobertura Mínima
- Todos os endpoints com sucesso (happy path)
- Todos os endpoints com input inválido (400)
- Todos os endpoints com ID inexistente (404)
- Endpoint de mover card (lógica mais complexa)
