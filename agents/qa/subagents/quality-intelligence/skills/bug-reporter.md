# Skill — bug-reporter

## Objetivo
Documentar bugs com contexto completo para reprodução e correção.

## Formato de Bug Report
```markdown
## Bug — [Título curto e descritivo]

**ID:** BUG-001
**Severidade:** Critical | High | Medium | Low
**Componente:** Board | Card | Column | API | Persistência

**Passos para reproduzir:**
1. ...
2. ...
3. ...

**Resultado esperado:** ...
**Resultado obtido:** ...
**Ambiente:** Browser, OS, resolução
**Evidência:** screenshot/video/log

**Possível causa:** ...
```
