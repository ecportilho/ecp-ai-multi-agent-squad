# Skill — session-based-tester

## Objetivo
Conduzir e documentar sessões de teste exploratório baseadas em tempo (Bach).

## Estrutura da Sessão
- **Duração:** 60-90 minutos
- **Charter:** definido antes da sessão
- **Notas:** em tempo real durante a sessão
- **Debriefing:** 15 min após para sintetizar achados

## Output
```json
{
  "charter": "...",
  "duracao_min": 60,
  "bugs_encontrados": [],
  "riscos_identificados": [],
  "areas_nao_cobertas": [],
  "notas": "..."
}
```
