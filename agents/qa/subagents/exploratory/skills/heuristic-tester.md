# Skill — heuristic-tester

## Objetivo
Aplicar heurísticas de teste para guiar a exploração (Bach — SFDPOT, HICCUPPS).

## Heurísticas SFDPOT aplicadas ao o produto
- **S**tructure: estrutura do JSON após operações complexas
- **F**unction: todas as funções do backlog executam corretamente?
- **D**ata: dados inválidos, nulos, muito longos, caracteres especiais
- **P**latform: Chrome, Firefox, Safari, mobile
- **O**perations: fluxos completos ponta a ponta
- **T**ime: operações rápidas em sequência, timeout de resposta
