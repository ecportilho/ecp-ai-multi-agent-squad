# Skill — edge-case-finder

## Objetivo
Identificar comportamentos inesperados e casos limite no o produto.

## Edge Cases do o produto
- Board sem colunas
- Coluna sem cards
- Título de card com 500+ caracteres
- Mover card para a mesma posição
- Deletar coluna com cards (o que acontece aos cards?)
- Board com 50+ colunas
- Caracteres especiais e emojis em títulos
- Duas abas do browser editando o mesmo board
