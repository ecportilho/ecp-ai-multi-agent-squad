# Skill — charter-designer

## Objetivo
Criar charters de sessões exploratórias com missão e escopo definidos.

## Formato de Charter
```
Explorar [área/funcionalidade]
Com [recursos/ferramentas]
Para descobrir [tipo de informação/problema]
```

## Charters do o produto
- "Explorar o drag-and-drop de cards entre colunas para descobrir falhas de posicionamento"
- "Explorar a persistência JSON com operações rápidas para descobrir condições de corrida"
- "Explorar o o produto em mobile para descobrir problemas de usabilidade em touch"
