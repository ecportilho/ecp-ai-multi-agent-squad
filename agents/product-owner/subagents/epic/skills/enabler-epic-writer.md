## Objetivo
Redigir épicos de habilitação técnica e arquitetural necessários para viabilizar épicos de negócio.

## Input
```json
{ "necessidade_tecnica": "...", "epico_de_negocio_relacionado": "..." }
```

## Output
```json
{
  "id": "EP-EN-001",
  "tipo": "arquitetura | infraestrutura | exploração | compliance",
  "titulo": "...",
  "descricao": "...",
  "valor_tecnico": "...",
  "epico_negocio_habilitado": "..."
}
```
