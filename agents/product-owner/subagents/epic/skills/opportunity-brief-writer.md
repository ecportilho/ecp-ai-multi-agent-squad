# Skill — opportunity-brief-writer

## Objetivo
Documentar cada oportunidade priorizada com contexto, evidências e outcome esperado.

## Input
```json
{ "oportunidade": "...", "evidencias": [], "kr_alvo": "..." }
```

## Output
```json
{
  "titulo": "...",
  "problema": "...",
  "quem_e_afetado": "...",
  "evidencias": [],
  "outcome_esperado": "...",
  "kr_que_move": "...",
  "o_que_nao_e_escopo": "..."
}
```
