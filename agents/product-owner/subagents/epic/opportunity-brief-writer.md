# Skill: opportunity-brief-writer

## Objetivo
Documentar cada oportunidade priorizada com contexto suficiente para o Designer trabalhar.

## Input
```json
{ "opportunity": {}, "evidence": {}, "kr": {} }
```

## Output
```json
{
  "opportunity_brief": {
    "opp_id": "",
    "title": "",
    "context": "situação atual do usuário",
    "opportunity": "o que o usuário precisa/quer/sofre",
    "evidence": [],
    "kr_connection": "",
    "success_looks_like": "como saberemos que resolvemos?",
    "out_of_scope": []
  }
}
```
