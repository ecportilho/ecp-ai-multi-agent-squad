# Skill — enabler-feature-writer (SAFe)

## Objetivo
Redigir features de habilitação (spike, infraestrutura, arquitetura) para o program backlog.

## Input
```json
{ "necessidade_tecnica": "...", "feature_negocio_relacionada": "..." }
```

## Output
```json
{
  "id": "FT-EN-001",
  "tipo": "spike | arquitetura | infraestrutura",
  "titulo": "...",
  "descricao": "...",
  "criterio_de_done": "...",
  "feature_habilitada": "..."
}
```
