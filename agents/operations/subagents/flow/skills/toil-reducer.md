# Skill: toil-reducer

## Objetivo
Identificar e automatizar trabalho manual repetitivo no pipeline GitHub Actions + Vercel.

## Métricas / Estratégias
- Automação de setup local via script `setup.mjs` (Node.js, cross-platform)
- Dependabot para atualizações automáticas de dependências
- Vercel bot para preview deployments automáticos em PRs
- Biome autofix em pre-commit hook
