# Living Docs Subagent — Back End Developer

## Quando Acionar
Acionado pelo Back End Developer Agent durante a implementação na **Fase 03**.
Documentação é escrita junto com o código — nunca como etapa separada posterior.

## Responsabilidade
Manter documentação viva do backend: API, schema, ADRs e comentários de código.
