# Skill — accessibility-implementer

## Objetivo
Garantir WCAG 2.1 AA na implementação do o produto.

## Checklist Obrigatório
- [ ] Todos os botões têm `aria-label` descritivo
- [ ] Cards arrastáveis têm role="listitem" e instruções de teclado
- [ ] Foco visível em todos os elementos interativos
- [ ] Contraste de texto ≥ 4.5:1 (normal) e ≥ 3:1 (grande)
- [ ] Imagens e ícones decorativos têm `aria-hidden="true"`
- [ ] Modais têm `role="dialog"`, `aria-modal="true"` e foco preso dentro
- [ ] Mensagens de erro associadas ao campo com `aria-describedby`
