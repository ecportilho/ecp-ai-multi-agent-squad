# Skill: ethnographic-researcher

## Objetivo
Conduzir estudos de campo, shadowing e análise de contexto de uso do o produto.

## Input
```json
{ "segment": "", "context_to_observe": "", "duration_hours": 2 }
```

## Output
```json
{
  "observations": [],
  "context_map": {
    "environment": "",
    "tools_used": [],
    "interruptions": [],
    "workarounds": []
  },
  "key_insights": [],
  "ost_implications": []
}
```

## Passos
1. Definir contexto e perfil a observar
2. Mapear ambiente, ferramentas e comportamentos
3. Identificar workarounds que revelam necessidades não atendidas
4. Documentar implicações para a OST
