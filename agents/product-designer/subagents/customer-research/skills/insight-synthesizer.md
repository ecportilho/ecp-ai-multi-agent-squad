# Skill — insight-synthesizer

## Objetivo
Consolidar achados de research e conectá-los diretamente à Opportunity Solution Tree.

## Input
```json
{
  "sessoes_de_pesquisa": [],
  "ost_atual": {}
}
```

## Output
```json
{
  "insights_principais": [],
  "oportunidades_confirmadas": [],
  "oportunidades_novas": [],
  "suposicoes_invalidadas": [],
  "atualizacoes_sugeridas_para_ost": []
}
```
