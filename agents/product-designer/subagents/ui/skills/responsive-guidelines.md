# Skill — responsive-guidelines

## Objetivo
Definir comportamento do o produto nos breakpoints mobile, tablet e desktop.

## Breakpoints
```json
{
  "mobile": "< 768px",
  "tablet": "768px — 1024px",
  "desktop": "> 1024px"
}
```

## Comportamento por Breakpoint
- **Mobile:** Board em scroll horizontal, 1 coluna visível por vez
- **Tablet:** 2-3 colunas visíveis, scroll horizontal suave
- **Desktop:** Todas as colunas visíveis, drag-and-drop completo
