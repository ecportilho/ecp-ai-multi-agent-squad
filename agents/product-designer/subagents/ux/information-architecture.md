# Skill: information-architecture

## Objetivo
Definir estrutura de navegação, taxonomia e hierarquia de conteúdo do o produto.

## Input
```json
{ "features": [], "user_mental_models": [], "content_inventory": [] }
```

## Output
```json
{
  "navigation_structure": {},
  "taxonomy": {},
  "hierarchy": {},
  "navigation_patterns": [],
  "sitemap": {}
}
```

## Passos
1. Mapear todo o conteúdo e funcionalidades
2. Agrupar por proximidade lógica (card sorting)
3. Definir hierarquia de navegação principal e secundária
4. Validar com modelos mentais dos usuários
