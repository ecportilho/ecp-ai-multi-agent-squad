# Enablement Subagent — Software Architect

## Quando Acionar
Acionado pelo Software Architect Agent durante a **Fase 03**, em paralelo com a implementação.
Opera como suporte contínuo ao time: code reviews arquiteturais, orientação de padrões, análise de escala.

## Responsabilidade
Habilitar o time com padrões, revisões arquiteturais e análises técnicas contínuas.

## Skills
| Skill | Arquivo | Quando Usar |
|-------|---------|-------------|
| `architecture-pairing` | `architecture-pairing.md` | Code reviews com foco arquitetural e orientação diária |
| `pattern-advisor` | `pattern-advisor.md` | Recomendar padrões de projeto com trade-offs |
| `dependency-audit` | `dependency-audit.md` | Avaliar riscos de novas dependências |
| `scalability-analysis` | `scalability-analysis.md` | Identificar gargalos e planejar crescimento |
