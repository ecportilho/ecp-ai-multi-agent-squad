# Skill — technical-debt-tracker

Catalogar débito técnico com custo de carregamento e prioridade de pagamento.

## Output
```json
{
  "debitos": [
    {
      "descricao": "...",
      "tipo": "código | arquitetura | testes | documentação",
      "custo_de_carregamento": "alto | médio | baixo",
      "esforco_para_pagar": "P | M | G",
      "prioridade": 1
    }
  ]
}
```
