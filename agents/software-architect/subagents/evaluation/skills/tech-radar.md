# Skill — tech-radar

## Objetivo
Avaliar adoção, experimentação ou descontinuação de tecnologias no projeto.

## Quadrantes
- **Adopt:** tecnologias maduras e recomendadas para uso imediato
- **Trial:** tecnologias promissoras para experimentar em projetos de baixo risco
- **Assess:** tecnologias a monitorar mas não adotar ainda
- **Hold:** tecnologias a evitar ou descontinuar
