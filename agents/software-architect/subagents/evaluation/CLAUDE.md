# Evaluation Subagent — Software Architect

## Quando Acionar
Acionado pelo Software Architect Agent ao tomar decisões técnicas de alto impacto na **Fase 03**,
ou ao avaliar novas tecnologias e débito técnico em qualquer fase.

## Responsabilidade
Avaliar tecnologias, riscos, decisões de build/buy e débito técnico com evidências.

## Skills
| Skill | Arquivo | Quando Usar |
|-------|---------|-------------|
| `tech-radar` | `tech-radar.md` | Posicionar tecnologias em adopt/trial/assess/hold |
| `risk-assessment` | `risk-assessment.md` | Identificar riscos técnicos e mitigações |
| `build-vs-buy` | `build-vs-buy.md` | Analisar custo/risco de construir vs comprar |
| `technical-debt-tracker` | `technical-debt-tracker.md` | Catalogar débito técnico com prioridade |
