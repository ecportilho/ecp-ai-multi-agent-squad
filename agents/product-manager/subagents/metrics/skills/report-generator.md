# Skill — report-generator

## Objetivo
Gerar relatórios de produto com foco em outcomes, não em outputs ou features entregues.

## Input
```json
{
  "periodo": "...",
  "okrs": [],
  "metricas_atuais": {},
  "audience": "executivo | time | stakeholders"
}
```

## Output
```json
{
  "sumario_executivo": "...",
  "progresso_por_kr": [],
  "insights": [],
  "proximos_passos": [],
  "formato": "markdown"
}
```
