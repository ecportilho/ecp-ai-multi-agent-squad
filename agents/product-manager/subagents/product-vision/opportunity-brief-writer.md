# Skill: opportunity-brief-writer

## Objetivo
Documentar o contexto estratégico de forma que todos os agentes do squad compartilhem o mesmo entendimento do problema.

## Input
```json
{ "vision": "", "okrs": [], "ost": {} }
```

## Output
```json
{
  "opportunity_brief": {
    "context": "situação atual do mercado e do produto",
    "problem": "problema central a ser resolvido",
    "target_outcome": "outcome esperado (não solução)",
    "success_metrics": [],
    "constraints": [],
    "out_of_scope": []
  }
}
```

## Passos
1. Sintetizar o contexto de mercado em 3-5 frases
2. Articular o problema central sem sugerir solução
3. Definir o outcome esperado em termos mensuráveis
4. Listar restrições conhecidas e o que está fora do escopo
