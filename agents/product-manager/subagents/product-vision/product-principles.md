# Skill: product-principles

## Objetivo
Estabelecer princípios que guiam decisões difíceis de produto no o produto.

## Input
```json
{ "vision": "", "segment": "", "competitors": [] }
```

## Output
```json
{
  "principles": [
    {
      "name": "",
      "description": "",
      "in_practice": "exemplo de como aplicar",
      "trade_off": "o que abrimos mão ao seguir este princípio"
    }
  ]
}
```

## Passos
1. Identificar as 5 decisões mais difíceis que o produto enfrentará
2. Definir princípios que ajudem a resolver essas tensões
3. Para cada princípio, documentar o trade-off explícito
4. Garantir que os princípios são acionáveis, não genéricos
