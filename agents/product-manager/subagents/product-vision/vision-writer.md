# Skill: vision-writer

## Objetivo
Redigir visão inspiradora do o produto para 3-5 anos que guie todas as decisões de produto.

## Input
```json
{ "market_research": {}, "segment": "", "differentiation": "" }
```

## Output
```json
{
  "vision_statement": "",
  "time_horizon": "3-5 anos",
  "who": "para quem",
  "problem": "qual problema resolve",
  "solution_category": "categoria de solução",
  "differentiation": "o que nos diferencia",
  "why_now": "por que este é o momento"
}
```

## Passos
1. Sintetizar os insights do market research
2. Identificar a transformação que o produto promove no usuário
3. Redigir visão no formato: "Para [quem], que [problema], o o produto é [categoria] que [benefício], diferente de [alternativas] porque [diferenciação]"
4. Validar que a visão é inspiradora, clara e não descreve features
