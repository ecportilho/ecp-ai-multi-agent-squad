# Skill — opportunity-brief-writer

## Objetivo
Documentar o problema central, contexto e outcome esperado — sem prescrever solução.

## Input
```json
{ "oportunidade": "...", "segmento": "...", "evidencias": [] }
```

## Output
```json
{
  "problema": "...",
  "contexto": "...",
  "quem_e_afetado": "...",
  "evidencias": [],
  "outcome_esperado": "...",
  "o_que_nao_e_escopo": "..."
}
```

## Regra
Nunca inclua soluções ou features. O brief descreve o problema, não a resposta.
