# Skill — product-principles

## Objetivo
Estabelecer princípios que guiam decisões de produto quando há conflito ou ambiguidade.

## Input
```json
{ "visao": "...", "segmento": "...", "valores_da_empresa": [] }
```

## Output
```json
{
  "principios": [
    { "principio": "...", "significado": "...", "exemplo_de_uso": "..." }
  ]
}
```

## Passos
1. Identificar os principais trade-offs do produto
2. Formular princípios que resolvem esses trade-offs com clareza
3. Validar que cada princípio é acionável em decisões reais
4. Garantir que os princípios são consistentes com a visão
