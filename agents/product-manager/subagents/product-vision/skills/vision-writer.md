# Skill — vision-writer

## Objetivo
Redigir a visão inspiradora de produto para 3-5 anos.

## Input
```json
{ "contexto_mercado": "...", "segmento_prioritario": "...", "diferenciacao": "..." }
```

## Output
```json
{ "visao": "frase inspiradora em 1-2 linhas", "narrativa": "parágrafo expandido" }
```

## Passos
1. Entender o segmento e o problema central
2. Articular o futuro desejado em linguagem inspiradora
3. Garantir que a visão orienta decisões sem prescrever soluções
4. Validar que é alcançável em 3-5 anos
