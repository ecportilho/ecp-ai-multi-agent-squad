# Skill — competitor-analysis

## Objetivo
Mapear concorrentes, seus posicionamentos e identificar gaps de mercado.

## Input
```json
{ "produto": "...", "segmento": "...", "concorrentes_conhecidos": [] }
```

## Output
```json
{
  "concorrentes": [
    {
      "nome": "...", "posicionamento": "...",
      "forcas": [], "fraquezas": [], "gaps": []
    }
  ],
  "oportunidades_diferenciacao": []
}
```

## Passos
1. Identificar concorrentes diretos e indiretos
2. Analisar posicionamento e proposta de valor de cada um
3. Mapear forças e fraquezas
4. Identificar gaps não atendidos pelo mercado
5. Sugerir oportunidades de diferenciação
