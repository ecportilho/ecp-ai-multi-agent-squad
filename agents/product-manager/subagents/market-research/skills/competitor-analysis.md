# Skill: competitor-analysis

## Objetivo
Mapear concorrentes diretos e indiretos, identificar gaps e oportunidades de posicionamento.

## Input
```json
{ "product": "o produto", "segment": "string", "depth": "superficial | detalhado" }
```

## Output
```json
{
  "competitors": [
    {
      "name": "",
      "type": "direct | indirect",
      "strengths": [],
      "weaknesses": [],
      "positioning": ""
    }
  ],
  "market_gaps": [],
  "differentiation_opportunities": []
}
```

## Passos
1. Listar concorrentes diretos (Trello, Notion, Asana, Linear)
2. Listar concorrentes indiretos (planilhas, post-its)
3. Analisar forças e fraquezas de cada um
4. Identificar gaps não atendidos
5. Mapear oportunidades de diferenciação
