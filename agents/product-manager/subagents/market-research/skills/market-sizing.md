# Skill — market-sizing

## Objetivo
Estimar o tamanho do mercado endereçável (TAM/SAM/SOM) e o potencial da oportunidade.

## Input
```json
{ "produto": "...", "segmento": "...", "geografia": "..." }
```

## Output
```json
{
  "tam": "...", "sam": "...", "som": "...",
  "crescimento_anual": "...", "fontes": []
}
```

## Passos
1. Identificar o mercado total endereçável (TAM)
2. Segmentar o mercado acessível (SAM)
3. Estimar fatia realista (SOM) para o produto
4. Projetar taxa de crescimento do mercado
5. Documentar fontes e premissas utilizadas
