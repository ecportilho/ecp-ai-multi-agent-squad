# Skill — trend-scanning

## Objetivo
Identificar movimentos de mercado, tecnologia e comportamento do consumidor relevantes.

## Input
```json
{ "dominio": "...", "horizonte": "6 meses | 1 ano | 3 anos" }
```

## Output
```json
{
  "tendencias": [
    { "descricao": "...", "impacto": "alto | médio | baixo", "urgencia": "..." }
  ],
  "ameacas": [], "oportunidades": []
}
```

## Passos
1. Mapear tendências tecnológicas relevantes ao domínio
2. Identificar mudanças no comportamento do consumidor
3. Avaliar impacto e urgência de cada tendência
4. Separar oportunidades de ameaças
